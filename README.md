# 🛍️ Customer Segmentation using K-Means Clustering

> Unsupervised machine learning project to segment mall customers based on their spending behavior.

## 📊 Overview
This project analyzes customer data from a mall and groups them into 5 distinct segments using K-Means clustering algorithm.

## 📁 Dataset
- **Source:** Mall Customers Dataset
- **Records:** 200 customers
- **Features:** Age, Annual Income, Spending Score, Gender

## 🔍 Customer Segments Found

| Segment | Income | Spending | Description |
|---------|--------|----------|-------------|
| 💰 Careful Spenders | High | High | Best customers |
| 🛍️ Impulsive | High | Low | Need targeting |
| ⭐ Standard | Low | High | Enthusiastic |
| 💡 Budget | Low | Low | Cost-conscious |
| 🎯 High Value | Medium | Medium | Average customers |

## 🛠️ Technologies Used
- Python
- Pandas & NumPy
- Scikit-learn
- Matplotlib & Seaborn
- Plotly (Interactive Dashboard)
- Google Colab

## 📈 Results
- Applied **Elbow Method** to find optimal number of clusters (k=5)
- Built an **interactive dashboard** with 4 visualizations
- Identified clear customer segments for business decisions
